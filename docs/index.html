Sillycard

本地 PNG 角色卡（Silly Tavern 规范）·

工具

产品说明

Sillycard 面向使用 Silly Tavern 生态的角色卡作者与收藏者：在设备上打开含角色卡元数据的 PNG，按与 Silly Tavern 主程序一致的解析规则读取内嵌 JSON（ccv3 优先于 chara），进行浏览、预览常用字段与嵌入世界书、编辑并写回 PNG。

应用侧重本地文件与离线工作流：你的清单、标签筛选与界面状态保存在本应用数据区；角色卡正文和元数据以你选择或保存在本机/外置存储中的 PNG 文件为准。应用不提供云端同步、账户体系或内置大语言模型生成角色内容。

功能概要

  • 浏览与整理本地 PNG 角色卡（含多库、归档与标签筛选）。
  • 预览：对标 Silly Tavern 常见展示字段与嵌入世界书结构。
  • 编辑：在结构化视图与 JSON 文本间切换；保存时写回 PNG 内嵌元数据（符合上述 chara / ccv3 约定）。
  • 支持浅色 / 深色 / 跟随系统外观（随系统实现而定）。

隐私与数据

以下为 Sillycard 在处理你的文件与设备数据时的说明。若你通过系统浏览器打开第三方网站，须遵守相应站点的条款与隐私政策。

我们处理哪些数据

  • 你在应用中打开、导入或编辑的 PNG 角色卡文件：其内容与元数据由应用在本机读写，用于展示与编辑。
  • 应用为运行而保存在沙盒内的偏好设置、库索引缓存、界面状态等，用于恢复你的使用体验。
  • 本应用不会主动采集可用于广告定向的个人画像数据，亦不要求注册账户即可完成核心功能。

我们如何使用这些数据

  • 仅用于在设备上呈现与管理你选择接触的角色卡文件，以及维护应用本地配置。
  • 开发者不运营用于接收或长期存储你角色卡内容的自有后端；默认情况下不向开发者服务器上传你的 PNG 或其中 JSON。

是否与第三方共享

  • 不会因使用 Sillycard 核心功能而将你的角色卡内容作为默认行为传输给开发者或广告商。
  • 若你主动使用系统共享、隔空投送、文件 App、iCloud 云盘等 Apple 或第三方服务同步文件，数据去向由你使用的系统与服务决定，不再属于本应用可控范围。

保留期限

  • 设备上：沙盒内数据随应用卸载或你清除应用数据而移除；PNG 文件本身保留在你保存的位置，直至你自行删除。

你的选择与撤回

  • 你可随时停止使用本应用、删除应用或移除已授予的文件/文件夹访问权限（在系统设置中）。
  • 请确保你对编辑与分享的内容拥有合法权利，勿利用本应用处理违法或侵权材料。

系统与说明

  • 适用于 macOS, iOS 双平台。
  • 角色卡需为符合 Silly Tavern 惯例的 PNG 格式；解析规则与上游 character-card-parser 对齐意图一致（应用版本可能随上游演进）。
